# Items Booking Service  (REST-API)
Application for booking items. With http-request filtering gateway.

### Technologies: ###
---
- Java, Spring Boot
- JPA (Hibernate), PostgreSQL
- Docker-compose (Multi-module project)
- Postman, JUnit, Mockito
- Maven
